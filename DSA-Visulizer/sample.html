Bubble Sort

Definition: Bubble Sort repeatedly steps through the list, compares adjacent elements and swaps them if they are in the wrong order.

Working: After each pass, the largest unsorted element "bubbles" to the end of the list.

Time Complexity: O(n²) average and worst case

Space Complexity: O(1)

#include using namespace std; // An optimized version of Bubble Sort void bubbleSort(vector& arr) { int n = arr.size(); bool swapped; for (int i = 0; i < n - 1; i++) { swapped=false; for (int j=0; j < n - i - 1; j++) { if (arr[j]> arr[j + 1]) { swap(arr[j], arr[j + 1]); swapped = true; } } // If no two elements were swapped, then break if (!swapped) break; } } // Function to print a vector void printVector(const vector& arr) { for (int num : arr) cout << " " << num; } int main() { vector arr = { 64, 34, 25, 12, 22, 11, 90 }; bubbleSort(arr); cout << "Sorted array: \n" ; printVector(arr); return 0; }
